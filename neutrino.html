Neutrino Oscillation Project

This is a research project on 2-flavor solar neutrino oscillation. I designed and implemented a modular numerical framework to simulate neutrino physics phenomena.

Project Details and Code

  • Project Title: 2-flavor Solar Neutrino Oscillation
  • Project Goal: Built a modular numerical framework capable of handling different physical quantities and validating through consistency checks.
